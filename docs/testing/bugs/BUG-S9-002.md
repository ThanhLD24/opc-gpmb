# BUG-S9-002 — HoSo workflow tree thiếu is_parallel, planned_start_date, planned_end_date

**Date:** 2026-05-05
**Reporter:** TQE (Sprint 9 Testing)
**Severity:** P2
**Component:** Backend `task.py` — `node_to_dict_with_tasks()`

---

## Symptom

Endpoint `GET /ho-so/{id}/tasks` trả về workflow tree của hồ sơ (dạng tree với nodes + tasks). Tuy nhiên, mỗi node trong tree **thiếu 3 fields Sprint 9**:
- `is_parallel` — cần thiết để FE hiển thị badge "Song song"
- `planned_start_date` — ngày bắt đầu kế hoạch của node
- `planned_end_date` — ngày kết thúc kế hoạch của node

**Test case thất bại:** TC-S9-17b

**Reproduce:**
```
GET /api/v1/ho-so/{hs_id}/tasks
→ tree[0].keys() = ['id', 'source_node_id', 'parent_id', 'code', 'name', 'level', 'order', 
                     'planned_days', 'per_household', 'require_scan', ...]
→ 'is_parallel' NOT IN keys  ← BUG
→ 'planned_start_date' NOT IN keys  ← BUG
→ 'planned_end_date' NOT IN keys  ← BUG
```

---

## Root Cause

**File:** `backend/app/api/v1/task.py`
**Function:** `node_to_dict_with_tasks()` (lines 166–189)

Hàm này serialize `HoSoWorkflowNode` thành dict nhưng không include 3 fields mới thêm trong Sprint 9:

```python
def node_to_dict_with_tasks(node: HoSoWorkflowNode, tasks: List[TaskInstance]) -> Dict[str, Any]:
    return {
        ...
        "per_household": node.per_household,   # ← existing
        # MISSING: "is_parallel": node.is_parallel,
        # MISSING: "planned_start_date": ...,
        # MISSING: "planned_end_date": ...,
        ...
    }
```

**So sánh:** `workflow.py` đã có `is_parallel` trong `node_to_dict()` (template serialization). Nhưng `task.py` dùng hàm riêng `node_to_dict_with_tasks()` cho HoSoWorkflowNode (snapshot) và chưa được cập nhật.

---

## Impact

- FE tab "Quy trình & Tiến độ" trong hồ sơ detail **không thể hiển thị badge "Song song"** vì `is_parallel` missing
- FE không thể hiển thị `planned_start_date` / `planned_end_date` per-node trong cây workflow nếu muốn
- Severity P2 vì ảnh hưởng đến UI workflow visualization

---

## Fix

**File:** `backend/app/api/v1/task.py`
**Function:** `node_to_dict_with_tasks()` — thêm 3 fields:

```python
"is_parallel": node.is_parallel,
"planned_start_date": node.planned_start_date.isoformat() if node.planned_start_date else None,
"planned_end_date": node.planned_end_date.isoformat() if node.planned_end_date else None,
```

Thêm sau dòng `"per_household": node.per_household`.
